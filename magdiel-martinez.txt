    1. What is Semantic HTML? 
    Semantic HTML is meaningful HTML that helps us understand our markup and what it's meant to do.  
    As opposed to using a bunch of <div>s and <p>'s we can use <section> and <nav>.
    
    2. What is HTML used for? 
    Hyper text markup is used primarily for giving structure and making the base layout or skeleton for our websites.
    As opposed to css which is meant to purely style the sheet.

    3. What is an attribute and where do we put it? 
    Attributes are meant to convey more information about our HTML usually used to include images and links, ex (href="https://image.com")

    4. What is the h1 tag used for? How many times should I use it on a page?
    H1 should only be used once on a page, for the main title as it is the biggest and most important Header we have.

    5. Name two tags that have required attributes
    The <input> and <img> tags will not display or function without their required attributes.

    6. What do we put in the head of our HTML document? 
    We put information and things we want to include in our HTML but that the users don't need to see, like our title, or styles for example.

    7. What is an id? 
    and ID is similar to a class, as it can be used to select elements. But it is unique and much more specific, only one can be used per tag.

    8. What elements can I add an id to? 
    You can add an ID to any element, but as you can only use it once it should be reserved for one important element that you would like to change.
    Like a header, h1 for example or if you want to link to another part of the page an ID would be used for that aswell.

    9. How many times can I use the same id on a page? 
    You can only use the same ID once per page

    10. What is a class? 
    A class is a css selector that can style the chosen elements.

    11. What elements can I add a class to? 
    You can add a class to any element.

    12. How many times can I use the same class on a page?

    SEO or if the image can not display it could show the ALT text on the page as a backup.
